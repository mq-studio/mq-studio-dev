# Website Archive

This directory contains historical artifacts and documentation from website development.

## Directory Structure

- **test-artifacts/** - Archived test results and Playwright reports
- **iterations/** - Completed iteration reports and phase documentation
- **sessions/** - Historical session summaries and completion reports
- **validation-reports/** - Old validation and testing reports

## Purpose

Artifacts are archived when they:
- Represent completed iterations or features
- Are test results from past runs (regenerable)
- Provide historical context
- Should be preserved but not clutter active development

## Finding Archived Items

### Search
```bash
grep -r "keyword" archive/
```

### Browse
```bash
ls archive/iterations/
ls archive/test-artifacts/
```

## Retention Policy

- **test-artifacts:** Keep for 6 months
- **iterations:** Keep for 2 years
- **sessions:** Keep for 1 year
- **validation-reports:** Keep for 1 year
